# Backup CLI

## Help(所有指令)
```
>> backup help
幫助訊息:
指令:help => 幫助訊息
----------------------------------------
指令:list => 列出遠端的檔案
----------------------------------------
指令:config => 設定
指令:config username => 設定使用者帳號
  選項:
      1.--show => 顯示使用者帳號
指令:config password => 設定使用者密碼
指令:config remote => 設定遠端位置
  選項:
      1.--show => 顯示遠端位置
----------------------------------------
指令:reset => 重設設定
指令:reset username => 重設username
指令:reset password => 重設password
指令:reset remote => 重設remote
指令:reset . => 重設所有欄位(username, password, remote)
----------------------------------------
指令:upload => 上傳檔案
參數:
    1.filename => 上傳的檔案名稱
    2.rename => 遠端存檔的檔案名稱
選項:
    1.--force => 若遠端檔案存在會覆蓋上去
    2.--private => 將檔案上傳到私有空間
----------------------------------------
指令:download => 下載
參數:
    1.filename => 要下載的檔案名稱
    2.rename => 下載完成的檔案名稱
選項:
    1.--force => 若有同名的檔案會覆蓋
    2.--private => 下載private的檔案
----------------------------------------
指令:delete => 刪除遠端檔案
參數:
    1.filename => 刪除的檔案名稱
選項:
    1.--private => 刪除private的檔案
----------------------------------------
指令:rename => 修改遠端檔案名稱
參數:
    1.old_filename => 要修改的檔案名稱
    2.new_filename => 新的檔案名稱
選項:
    1.--private => 重新命名private的檔案
----------------------------------------
指令:move => 移動檔案
參數:
    1.filename => 要移動的檔案名稱
    2.destination => 要移動到的目的地
選項:
    1.-d => 移動資料夾
----------------------------------------
指令:publish => 發布網頁專案到伺服器上
參數:
    1.file => 要上傳的檔案(.為全選)
    2.path => 上傳到伺服器根目錄下的相對位置
選項:
    1.--force => 若遠端檔案存在會覆蓋上去
指令:publish list => 顯示伺服器相對位置下的檔案
  參數:
      1.directory => 要列出檔案的資料夾
指令:publish clear => 顯示伺服器相對位置下的檔案
  參數:
      1.directory => 要從伺服器清除的資料夾
----------------------------------------
指令:pull => 從伺服器相對位置下載專案
參數:
    1.directory => 要從伺服器下載的資料夾
----------------------------------------
```
